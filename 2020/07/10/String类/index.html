    String类

    String类的定义：public final class String

    这意味着String类不允许继承，方法不能被复写！

    String类的方法

    1.获取的方法

    1.获取字符串的长度,int length();

    2.根据位置获取位置上某个字符；

    3.根据字符获取该字符在字符串中第一次出现处的索引；

    int indexOf(int ch),这里的ch是传入字符的ascii码；

    int indexOf(int ch,int fromIndex),从指定位置fromIndex开始获取ch第一次出现的位置；

    int indexOf(String str),返回的是字符串str在字符串中第一次出现的位置；

    int indexOf(String str,int fromIndex),从指定位置fromIndex开始获取str第一次出现的位置.

    注意对于以上四种方法，倘若未能找寻到所寻找的字符或字符串的下标，则会返回-1.

    4.反向索引

    int lastIndex(int ch),返回ch在此字符串中最后一次出现处的索引；

    int lastIndexOf(int ch,int fromIndex)，返回指定字符在此字符串中最后一次出现处的索引；

    int lastIndexOf(String str)，返回指定子字符串在此字符串中最右边出现的索引；

    int lastIndexOf(String str,int fromIndex)，返回str在此字符串中从指定位置开始反向搜索的最后出现位置.

    2.判断的方法

    1.boolean startsWith(String suffix)，字符串是否以指定内容开头;

    boolean endsWith(String suffix)，字符串是否以指定位置结束；

    2.boolean isEmpty()，当length()为0时返回true,否则返回false;

    3.boolean contains(CharSequence cs)，当此字符串包含指定char值序列时，返回true

    (string 是接口CharSequence的一种实现形式);

    4.boolean equals(str)，判断内容是否相同，实际上是复写了Object类中的equals方法；

    boolean equalsIgnoreCase(str)，判断内容是否相同并忽略大小写;

    3.替换的方法

    1.String replace(char oldchar,char newchar)，将原串的所有oldchar字符全部替换为newchar；

    String replace(CharSequence target,CharSequence replacement),替换字符串；

    以上两种方法在找不到被替换的对象时，就直接返回原串.

    4.切割的方法

    String[] split(regex),以regex为分隔，将字符串分割，返回字符串数组；

    5.子串

    String substring(begin),从指定位置一直到结尾；
    String substring(begin,end),包含头部不包含尾部，获取下标为begin~(end-1)；

    6.转换，去除，比较

    1.String toUpperCase(),将字符串转换成大写；

    String toLowerCase(),将字符串转换成小写；

    2.String trim(),将字符串两端的多个空格去除；

    3.int compareTo(string),对两个字符串进行字典顺序的比较，按字典顺序将此String对象表示的字符序列与参数字符串所表示的字符序列进行比较，若按字典序此String对象位于参数字符串之前（后），则比较结果为一个负（正）整数，相等则为0;返回整数为第一个能够比较出大小的两个ascii码之差.

    4.char[] toCharArray(String s),将一个字符串转换成字符数组;

    5.String(char[])，将字符数组转成字符串；
    String(char[],offset,count),将字符数组中的一部分转成字符串；

    静态方法：

    static String copyValueOf(char[]);
    static String copyValueOf(char[] data,int offset,int count);
    static String valueOf(char[]),返回给定参数的原生 Number 对象值，参数可以是原生数据类型, String等。
