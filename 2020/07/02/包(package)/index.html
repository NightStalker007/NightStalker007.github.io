包(package)

阅读量:

功能

用于对类文件进行分类管理，给类提供多层命名空间，它也是一种封装形式。

位置

它写在程序文件的第一行，类名的全称是包名.类名。

在指定目录生成包：javac -d location(当前位置则为’.’) java文件名

包使类文件和源文件相分离，使用包中类时，路径指向包文件所在文件夹 set classpath=…

注意事项

1.编译时，倘若使用的包不在当前文件夹，同样需要设置路径到包所在文件夹。

2.在使用其他包的类时，需要注意的是，该类的权限必须是public,否则编译报错：class在package中不是公共的，无法从外部程序包中对其进行访问，同时，类公有后，被访问的成员也要公有才可以被访问。

定义包packb中的类DemoB

由public修饰的class中的public方法和数据可以被访问

3.编译时，类名前面同样需要注明包名，而一个java文件仅能出现一个公有类
