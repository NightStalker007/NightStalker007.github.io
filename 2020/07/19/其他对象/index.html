    System

    getProperties

    该方法返回一个Properties类，该类是Hashtable的子类，也就是Map集合的一个子类对象，那么可以通过map的方法取出该集合中的元素，该集合中存储的都是字符串，没有泛型定义.
    也可以使用专有方法Set stringPropertiesNames()来返回此属性列表的键集；
    可以通过方法static String setProperty(String key,String value),来设置指定键指示的系统属性；
    要获得指定信息使用getProperty()即可；

    寻找不到键则返回null.

    Runtime

    每个Java应用程序都有一个Runtime类实例，使应用程序能够与其运行的环境相连接，可以通过getRuntime方法获取当前运行时，应用程序不能创建自己的Runtime类实例
    static Runtime getRuntime(),返回与当前Java应用程序相关的运行时对象；Runtime类并没有提供构造函数，说明不可以new一个对象，该类中有静态方法有费静态方法，该类提供了本静态方法获取本类对象.(该类使用了单例设计模式完成)

    public Process exec(String command),在单独的进程中执行指定的字符串命令;
    destroy(),杀掉子进程；
